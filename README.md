# danbryan/homebrew-tap

Homebrew tap for [Attaché](https://attache.fm), the native macOS app
that speaks your AI agents' updates out loud.

## Install

```sh
brew install --cask danbryan/tap/attache
```

Requires macOS 13 (Ventura) or later on Apple Silicon.

## Update

```sh
brew upgrade --cask attache
```

## Uninstall

```sh
brew uninstall --cask attache
```
